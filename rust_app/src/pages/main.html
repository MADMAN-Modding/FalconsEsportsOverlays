Welcome to the Falcons Esports Overlays Controller!

Instructions

  • Go to the Overlay Files page to the overlays
  • Go to Config Page to set your settings
  • Go to the Web Server Page to start the website
  • To Change the overlay information you can control it from the Overlay Data page
  • The URL is http://127.0.0.1:8080 and the dimensions are 1920x1080
  • Make sure to add the overlay as a browser in OBS
  • You can change the falcon logo by going to the Config Page and pressing the image button
  • The enabled sports can be changed on the Config Page
