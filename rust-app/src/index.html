Welcome to the Falcons Esports Overlays Controller!

Instructions

  1. Make sure git is added to the PATH, you can test this with git -v in a command line
  2. Go to config to set your settings
  3. Go to Download & Update to get the overlay
  4. Go to Web Server to start the server
  5. Finally now you can control the overlay from the Overlay Data Page
  6. Make sure to add the overlay as a browser in OBS
  7. The URL is http://localhost:8080 and the dimensions are 1920x1080
  8. You can change the falcon image by going to the images folder in the overlay and changing Esports-Logo.png
